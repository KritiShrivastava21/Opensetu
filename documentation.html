OpenSetu: How It Works

OpenSetu is powered by DNS over HTTPS (DoH), a network protocol that resolves domain names securely over HTTPS. DoH enhances privacy and security by preventing intermediaries from spying on or tampering with DNS queries.

Bharat ka Internet

Bharat ka Internet is built on Handshake, an open-source, decentralized naming protocol that serves as an alternative root zone for the internet. It enables censorship-resistant, tamper-proof domain resolution ‚Äî the backbone of our internet.

In the near future, we also plan to support protocols like ENS (Ethereum Name Service) and IPFS (InterPlanetary File System) to enable a more open and interoperable web.
