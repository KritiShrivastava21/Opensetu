OpenSetu lets you access the internet in your mother tongue. No app needed. Just change your browser settings – and the web speaks your language.

It’s infrastructure that connects you to the language-specific versions of the web. Start using OpenSetu: pranam.opensetu.in

What Does
OpenSetu Do?

OpenSetu is a DNS resolver which allows you to access public goods and government utilities in your mother tongue. Just type the name of your domain in your own language and it will redirect it to the language version of the official site.

For example, if you are a Hindi speaker, you can visit pmkisan.gov.in by simply typing पीएमकिसान.सर.भा in Devanagari or if you use Tamil just type பிஎம்கிசான்.ப etc.

Disclaimer: OpenSetu, as of now, redirects only to websites that support language versions via URL. If a website uses JavaScript or cookies to manage language switching, users will need to manually change the language using the site’s language selector.
